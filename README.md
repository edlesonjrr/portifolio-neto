## ✨ Features

- **Loader screen** — terminal-style progress bar that gets stuck at 1%
- **Hero section** — two-column layout with photo, glitch name effect and animated stats
- **About** — cards with hidden skills reveal button
- **RPG Skills** — animated progress bars with hover tooltips
- **Experience timeline** — the legendary 1-day hospital job + army recruitment block
- **Mini game** — catch the falling boxes (keyboard & touch support)
- **Personal life** — relationship status modal, hobbies and daily routine
- **Life goals** — interactive checkboxes with mini confetti
- **Call to action** — confetti explosion on hire button
- **Easter eggs** — Konami Code, console messages, clickable photo dialogues

## 🚀 How to Run

No build tools needed. Just open `index.html` in your browser.

```bash
git clone https://github.com/your-username/netinho-portfolio.git
cd netinho-portfolio
# open index.html in your browser
```

Or use a local server:

```bash
npx serve .
# or
python -m http.server 8000
```
